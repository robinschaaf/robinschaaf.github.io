Full-Stack Developer

I've been in database/web development since graduating with my B.S. in Computer Technology from Purdue in 2001. Over the years, this has meant:
  • (The standards): HTML, Javascript (yes, pre-jQuery!), CSS
  • Perl / Postgresql - In my spare time I developed a user-content driven site, mailleartisans.org, which is still in use today. This was done in about 2002, well before myspace started or other user content driven sites. If only I'd picked something with broader appeal than chainmail! :-)
  • Coldfusion / Oracle, and some Java (while I worked for Discover Financial)
  • ETL Development, Data Mart/Data Warehousing with Ab Initio (also at Discover)
Since I've been at the library:
  • ETL and Report Development using JasperETL, Unix shell scripts and Perl for the custom built reporting front end against Oracle. This was later rewritten to MySQL for cost/maintainability reasons, and now I know all about repairing running MyISAM tables in a large data environment :-)
  • PHP / MySQL
  • Ruby on Rails / JQuery / CoffeeScript / SCSS / Microdata via schema.org
  • Mobile site development using a combination of Rails, JQuery Mobile and iOS and Android apps via PhoneGap

With a UX Obsession

I've always been a developer, but by far I'm best at figuring out how to use software to solve problems for users. It's what drove me to choose to get into computers in the first place - in my high school retail work we had to use a point-of-sale system which used the F5 key to finish the order - right next to the F4 key that completely wipe out the order, without verification first. (seriously, though, who would do this ?!) I knew that I could write better software than that.
